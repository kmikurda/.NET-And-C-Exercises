# 🚀 .NET / C# Mastery Series

> A structured, hands-on learning series covering modern .NET and C# — from language internals to production-grade patterns, frameworks, and performance engineering.

**This project was created for my own purposes. Each lesson has a 15-20 min effort. I hope it will be my daily routine task that improves my skills**

---

## 🎯 Who is this for?

- Developers who know C# basics and want to level up to mid/senior-level patterns
- Engineers looking to fill gaps in modern .NET (8/9/10) features
- Anyone who learns best through code-first, example-driven lessons

---

## 📐 Design Principles

Every lesson follows the same rules:

- ✅ **Clean, idiomatic code** — no shortcuts, no bad habits
- ✅ **Before / After** — shows the old way vs. the modern way
- ✅ **Production-relevant** — patterns used in real codebases, not toy examples
- ✅ **Self-contained** — each lesson stands on its own
- ✅ **SOLID, DRY, YAGNI** respected throughout
